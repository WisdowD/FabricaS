OBS 1:
Caso esteja dando erro de não encontrar nenhuma biblioteca sql tente instalar ela pelo comando:
pip install mysql-connector-python
se esse comando estiver sendo barrado pelo console, vá no powershell colocando ele como administrador
e coloque o comando: Set-ExecutionPolicy RemoteSigned
Digite S e confirme
